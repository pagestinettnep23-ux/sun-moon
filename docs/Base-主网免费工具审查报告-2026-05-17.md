### 3.2 TestnetUsdcAdapter reentrancy-balance

位置：

```text
contracts/hooks/TestnetUsdcAdapter.sol
swapFeeAssetToUSDT(...)
```

归类：

```text
测试网 Adapter，不是当前 Base 主网 dry-run 的直接 USDC-only 主路径。
合约自身已有 onlyAuthorizedHook、nonReentrant、router allowlist。
```

建议：

```text

如果未来要在主网启用任意代币换 USDC Adapter，必须重新设计并单独审计。
```

### 3.3 MoonCurve / SunCurve strict equality

位置：

```text
contracts/MoonCurve.sol
contracts/SunCurve.sol
lastMintBlock == block.number
```

归类：

```text
有意设计。
用途是阻止同一区块内 mint 后马上 burn 的路径，降低闪电交易/同块操纵风险。
不是依赖价格精确相等，也不是随机数逻辑。
```

验证：

```text
testMoonBurnRejectsSameBlockAfterMint
testSunBurnRejectsSameBlockAfterMint
```

建议：

```text
保留。
审计时说明这是同区块防护，不是危险业务判断。
```
