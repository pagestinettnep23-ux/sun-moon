# 本地环境

本文档记录当前电脑上用于本项目的开发工具状态。

## 已安装工具

### Git

用途：

- 保存代码版本。
- 后续安装合约依赖。
- 后续提交代码和回滚历史。

当前版本：

```text
git version 2.54.0.windows.1
```

### Foundry

用途：

- `forge`：编译和测试 Solidity 合约。
- `cast`：查询链上数据、调用合约。
- `anvil`：启动本地区块链。
- `chisel`：Solidity 交互式实验工具。

当前版本：

```text
forge 1.7.1
cast 1.7.1
anvil 1.7.1
```

安装位置：

```text
C:\Users\Administrator\.foundry\bin
```

## 项目依赖

合约依赖已安装到项目的 `lib/` 目录。

当前版本：

```text
forge-std v1.16.1
openzeppelin-contracts v5.6.1
prb-math v4.1.1
```

项目已生成：

```text
.gitmodules
foundry.lock
lib/forge-std
lib/openzeppelin-contracts
lib/prb-math
```

## PATH 说明

Git 和 Foundry 已加入环境变量。

如果在当前已经打开的 PowerShell 或 Codex 里直接输入 `forge` 识别不到，重新打开 PowerShell 或重启 Codex 即可。Windows 已打开的程序不会自动读取刚刚修改过的环境变量，这是正常现象。
