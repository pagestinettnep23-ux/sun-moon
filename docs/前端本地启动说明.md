# 前端本地启动说明

本文档说明如何启动 SUN/MOON 第一版本地链前端。

## 1. 前置条件

- Anvil 本地链正在运行：`http://127.0.0.1:8545`
- 已完成本地合约部署，地址参考：`docs/部署记录-Anvil-2026-05-12.md`
- 浏览器安装 MetaMask 或兼容钱包
- 钱包网络为 Anvil，本地 Chain ID：`31337`

## 2. 启动前端

进入前端目录：

```powershell
cd C:\Users\Administrator\OneDrive\文档\uniswap-v4-hook-project\frontend
```

启动本地服务：

```powershell
node server.mjs
```

打开页面：

```text
http://127.0.0.1:5173
```

## 3. 第一版支持的路径

买入路径：

```text
领取 MockUSDT -> 授权 USDT -> 铸造 SUN -> 授权 SUN -> 铸造 MOON
```

退出路径：

```text
卖出 MOON -> 授权 SUN -> 卖出 SUN
```

注意：当前最小版合约中，卖出 MOON 不需要提前授权 MOON。

## 4. 当前不做的事情

- 不连接主网。
- 不接入真实 USDT。
- 不做真实资金入口。
- 不做一键 `USDT -> MOON`。
- 不做一键 `MOON -> USDT`。
- 不做 AMM 加池入口。
- 不修改 Solidity 合约。
- 不修改部署脚本。
