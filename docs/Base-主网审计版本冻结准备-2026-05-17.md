# Base 主网审计版本冻结准备 - 2026-05-17

本文档用于把当前 SUN/MOON Base 主网候选版本整理成“可交给审计方的固定版本”。它不是主网部署批准，也不包含私钥。

当前仍然：

```text
不部署 Base 主网
不广播交易
不接真实资金
不索要私钥
```

## 1. 当前 git 状态

已检查：

```text
git 仓库存在
当前分支：master
当前还没有任何 commit
当前没有可引用的 HEAD commit
```

这意味着现在还不能把审计版本写成“commit hash = xxx”。审计前需要先创建第一个本地 commit，之后再打 tag。

## 2. 冻结版本目标

建议冻结名：

```text
commit message: prepare SUN MOON Base mainnet audit package
tag: audit-sun-moon-base-2026-05-17-rc1
```

含义：

```text
audit = 给审计用
sun-moon-base = SUN/MOON Base 方案
2026-05-17 = 冻结日期
rc1 = 第一个审计候选版本
```

如果审计后修改代码，应创建新的 commit 和新的 tag，例如：

```text
audit-sun-moon-base-2026-05-17-rc2
audit-sun-moon-base-2026-05-18-rc1
```

说明：

```text

out/、cache/、broadcast/ 是本地生成物
frontend 备份 HTML 和 tsbuildinfo 是本地过程文件
.env 可能包含密钥或 RPC，不允许进入版本库
```
