# Base 主网部署前最终红线清单 - 2026-05-20

本文是给技术小白使用的一页停止清单。

它不是主网部署批准，不是广播记录，也不是交易教程。

只要本清单里任何一条不满足，就停止，不要继续。

```text
不执行 Base 主网交易
不使用真实资金
不要求或记录私钥、助记词、恢复词
不使用 --private-key
不广播
```

## 1. 最高优先级红线

看到以下任意情况，立即停止：

```text
任何人要求你输入私钥
任何人要求你输入助记词
任何人要求你输入恢复词
任何人要求你粘贴钱包导出内容
任何命令包含 --private-key
任何命令包含 --broadcast
任何命令包含 cast send
任何步骤要求使用真实 ETH 或真实 USDC 做演练
任何页面弹出 Base 主网交易签名
任何脚本显示 broadcastRequested=true
任何脚本显示 executeRequested=true
任何脚本显示 privateKeyPresent=true
任何脚本显示 broadcastAllowed=true
任何脚本显示 executionBlocked=false
```

当前硬性要求仍然是：

```text
不执行 Base 主网交易。
不使用真实资金。
```

所以当前阶段只能做：

```text
文档
本地测试
只读检查
fork dry-run
Git 提交 / 推送 / 标签
```

## 2. 网络和钱包红线

看到以下任意情况，立即停止：

```text
钱包网络显示 Base mainnet，并且页面要求确认交易
钱包提示会消耗真实 gas
钱包提示会发送真实 ETH
钱包提示会转出真实 USDC
钱包提示签名内容看不懂
浏览器页面要求连接真实资金钱包做演练
```

允许继续的情况只有：

```text
命令是只读检查
命令是本地测试
命令是 fork dry-run
命令没有 --broadcast
命令没有 --private-key
没有钱包弹窗
没有真实资金提示
```

## 3. Git 版本红线

继续只读复核前，必须满足：

```text
git status 显示 main...origin/main
工作区没有未提交的 contracts/ 改动
工作区没有未提交的 script/ 主网脚本改动
工作区没有未提交的 test/ 主网测试改动
```

如果改过以下目录，必须停止并重新跑测试：

```text
contracts/
script/
test/
foundry.toml
remappings.txt
```

当前只读复核安全快照：

```text
commit = 46540d2 docs: add base mainnet readonly review record
tag = mainnet-readonly-review-2026-05-20
```

如果本地 HEAD 不是这个提交或更新后的明确安全提交，不要直接进入下一步。

## 5. 预测地址红线

这些值必须和最新只读复核记录一致：

```text
SUN_TOKEN
SUN_CURVE
MOON_TOKEN
MOON_CURVE
CREATE2_HOOK_DEPLOYER
HOOK_SALT
PREDICTED_HOOK
SUN_USDC_POOL_ID
MOON_USDC_POOL_ID
```

任何一个不同，停止。

特别注意：

```text
预测地址不是已部署地址。
poolId 不是已初始化证明。
dry-run 成功不是主网上线成功。
```

如果 `MAINNET_DEPLOYER nonce` 变化，必须停止并重新计算：

```text
核心合约地址
HOOK_SALT
PREDICTED_HOOK
两个 poolId
19 笔草案
```

## 6. Hook 路径红线

Base 主网路径只允许新版 Hook：

```text
BaseSunMoonUsdcFeeV4Hook
```

看到以下任意旧路径用于主网，立即停止：

```text
BaseMoonAmmFeeV4Hook
FindBaseMoonAmmFeeV4HookSalt
PrepareBaseSepoliaHookDeploy
TinyMoonUsdc
```

旧路径可以留在历史文档或测试里，但不能成为主网默认路径。

## 7. 只读脚本必须看到的安全字样

只读脚本输出必须包含：

```text
Script ran successfully
chainId = 8453
simulationOnly = true
broadcastRequested = false
```

19 笔草案脚本还必须看到：

```text
executeRequested = false
privateKeyPresent = false
broadcastAllowed = false
executionBlocked = true
moonAmmBindingTxIncluded = true
totalTransactionsPlanned = 19
```

如果任意一项不同，停止。

## 8. 测试红线

继续前，主网准备相关测试必须全部通过。

最低要求：

```text
Core dry-run preparation 通过
Hook salt preparation 通过
PoolId preparation 通过
Mainnet fork dry-run preparation 通过
BaseSunMoonUsdcFeeV4Hook 通过
19 笔广播草案测试通过
```

任何测试失败，停止。

不要用“只是文档小改”来跳过核心测试。

## 9. 绑定红线

19 笔草案必须包含：

```text
SunCurve.setMoonAMM(PREDICTED_HOOK)
```

真实部署后才可以做以下只读检查：

```text
SunCurve.moonAMM() == PREDICTED_HOOK
Hook.usdc() == USDC_TOKEN
Hook.sunCurve() == SUN_CURVE
Hook.protocolBudget() == PROTOCOL_BUDGET_WALLET
```

没有真实部署前，不允许把这些项打勾成“已链上通过”。

## 10. 最终一句话

当前阶段的安全结论是：

```text
可以继续做只读复核。
不可以执行 Base 主网交易。
不可以使用真实资金。
不可以广播。
```

如果不确定某一步是不是安全，只做一件事：

```text
停止，先问清楚。
```
