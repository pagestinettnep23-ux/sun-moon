SUN/MOON 前端预览方式

当前正确的本地前端地址：

http://127.0.0.1:4173/

简单步骤：

1. 双击 START_STATIC_SERVER.cmd，启动本地预览服务器。
2. 浏览器打开 http://127.0.0.1:4173/。
3. 页面应显示 Base Sepolia 测试网，并处于 read only 状态。

安全提醒：

- 这个前端当前只用于 Base Sepolia / 本地只读测试。
- 不要在这里使用真实资金。
- 不要输入或保存私钥、助记词、恢复词。
- 上主网前，必须单独更新并复核主网合约地址和 ABI/selector。
